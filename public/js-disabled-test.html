Testing Property Listings with JavaScript Disabled

This guide will help you test if the server-side property listings work correctly when JavaScript is disabled in the browser.

Disable JavaScript in your browser

Chrome: Open DevTools (F12) > Settings (⚙️) > Preferences > Debugger > Check "Disable JavaScript"

Firefox: Open DevTools (F12) > Settings (⚙️) > Debugger > Check "Disable JavaScript"

Safari: Safari menu > Preferences > Advanced > Check "Show Develop menu" > Develop menu > Disable JavaScript

Visit these property listings pages

Test the following URLs to see if they load properly with JavaScript disabled:

  • Commercial Offices for sale in Business Bay
  • Apartments for sale in Downtown Dubai
  • Shops for rent in Dubai Marina

Check for these key elements

Verify the following are working correctly:

  • Property listings are visible and properly formatted
  • Images are loading
  • Property details (price, beds, baths, size) are visible
  • Links to property detail pages work
  • Server-side pagination links work (if available)

Remember: Don't forget to re-enable JavaScript when you're done testing!
